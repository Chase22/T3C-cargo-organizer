# T3C Cargo Organiser Help

## How to Use

### Copy initial lend lease message
Right-click the lend-lease message by minerva and select `copy text`
![](/right.png)

### Insert into input field
Paste the entire message text into the left input field

### Click `Distribute Cargo` to start the process

### Copy the output
The output will show up on the right. Check it and copy it with the `copy` button at the top.

This can now be simply inserted into Discord
